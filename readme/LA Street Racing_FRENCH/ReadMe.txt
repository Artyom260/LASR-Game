2. Installation
=========================

Avant d'installer L.A. Street Racing, assurez-vous que tous les programmes d'arrière-plan sont fermés (anti-virus, client e-mail, navigateur Internet,logiciels de partage de fichiers, messagerie instantanée).  La majorité des problèmes lors de l'installation peuvent être évités
en fermant ces programmes.

Insérez le CD-Rom de L.A. Street Racing dans votre lecteur CD\DVD, cliquez sur le bouton INSTALLATION, et suivez les instructions affichées à l'écran pour terminer l'installation.

Si la fonction de notification d'insertion automatique est désactivée ou si l'écran d'installation ne s'affiche pas, effectuez les étapes suivantes :

1.	Cliquez sur démarrer.
2.	Cliquez sur Exécuter.
3.	Inscrivez D:\SETUP.EXE (où "D" est la lettre de votre lecteur CD\DVD).
4.	Suivez les instruction affichées à l'écran.

5. Dépannage
=========================

- Assurez-vous que votre ordinateur correspond à la configuration minimale requise. Installer L.A. Street Racing sur un ordinateur qui ne correspond pas à la configuration minimale requise peut engendrer des résultats imprévisibles.

- Avant d'installer le jeu, assurez-vous que tous les programmes d'arrière-plan tels que les logiciels d'anti-virus, sont fermés.  La majorité des problèmes lors de l'installation peuvent être évités en fermant ces programmes

Si vous rencontrez toujours des problèmes lors de l'installation de L.A. Street Racing, effectuez les étapes suivantes :

1. Avec le CD 1 inséré, double cliquez sur POSTE DE TRAVAIL.

2. Faites un clic droit sur le lecteur CD\DVD et sélectionnez EXPLORER.

3. Ouvrez le menu déroulant EDITION et sélectionnez l'option SÉLECTIONNER TOUT.

4. Ouvrez à nouveau le menu déroulant EDITION et sélectionnez COPIER.

5. Trouvez un endroit sur votre disque dur pour copier ces fichiers. Nous conseillons de les copier dans un dossier sur votre bureau. Pour ce faire    faites un clic droit sur votre bureau et sélectionnez NOUVEAU, puis DOSSIER.  Ouvrez le Nouveau Dossier créé, faites un clic droit dedans et sélectionnez COLLER.

6. Les fichiers vont maintenant se copier du CD 1 sur votre disque dur.  L'opération prendra quelques minutes.

7. Une fois que le jeu est copié, double cliquez sur RunSetup.exe à l'endroit où vous l'avez copié pour installer le jeu.

Ces étapes enlèvent le lecteur CD\DVD de la procédure d'installation, afin que tous les problèmes liés au disque soient évités.

- Si vous utilisez un système d'exploitation Windows Xp ou 2000 vous devez installer L.A. Street Racing avec un compte utilisateur qui possède les privilèges Administrateur ou Power User. Si vous n'avez pas les privilèges Administrateur ou Power User, veuillez contacter votre administrateur système
pour plus de détails.

- Si vous utilisez un Pare-feu ou un routeur, vous aurez peut-être quelques difficultés à jouer via un Réseau local (LAN) ou sur Internet.  Vous devez configurer votre pare-feu\routeur pour permettre à L.A. Street Racing d'accéder au ports 443, 12321, 12346 pour le serveur maître des matchs.

- Assurez-vous que les fonctions TCP et UDP sont activées pour tous les ports.  Si vous ne savez pas comment faire, contactez votre administrateur ou votre
revendeur de pare-feu\routeur pour des informations supplémentaires.

- Si vous utilisez Windows XP avec le Service Pack 1 (SP1) installé et que votre pare-feu de connexion Internet est activé vous rencontrerez peut-être quelques difficultés à jouer via un réseau local (LAN) ou sur Internet. Pour corriger cela, vous devez soit installer le Service Pack 2 (SP2), soit désactiver le pare-feu de connexion Internet depuis vos paramètres réseau. Effectuez les étapes suivantes pour désactiver votre pare-feu de connexion Internet :


1. Cliquez sur démarrer.

2. Sélectionnez Panneau de configuration ou, si vous utilisez Windows en mode classique, sélectionnez Paramètres.

3. Sélectionnez Connexion Internet et Connexions réseau puis Connexions réseau.  Si vous utilisez Windows en mode classique sélectionnez Connexions réseau.

4. Faites un clic droit sur Connexion au réseau local et sélectionnez Propriétés.

5. Cliquez sur l'onglet "Avancé".

6. Assurez-vous que la case "Protéger mon ordinateur et mon réseau en limitant ou en interdisant l'accès à cet ordinateur depuis internet" est décochée.

- Si vous utilisez Windows XP Service Pack 2 (SP2), vous aurez peut-être quelques difficultés à jouer à L.A. Street Racing via Internet notamment à cause du pare-feu Windows. Pour corriger cela, effectuez les étapes suivantes :

1. Lancez L.A. Street Racing.

2. Dans le menu principal, cliquez sur Paramètres multijoueur.

3. Quand la fenêtre des serveurs s'affiche, appuyez sur ALT et TAB en même temps. Cette action vous fera revenir sur le bureau Windows.

4. Une fenêtre "Alerte de Sécurité Windows" devrait s'afficher.  Cliquez sur Autoriser pour permettre à L.A. Street Racing  d'accéder aux serveurs de jeu Internet.

5. Cliquez sur L.A. Street Racing dans la barre de tâches Windows.  Vous pourrez ainsi revenir dans le jeu.

6. Quittez L.A. Street Racing en sélectionnant Quitter le jeu dans le menu principal.

7. Relancez L.A. Street Racing, vous devriez maintenant pouvoir vous connecter aux parties Internet sans problème.

- Certaines cartes graphiques, souvent celles installées dans les ordinateurs portables, ont une fonction de rotation qui peut forcer le jeu à s'afficher différemment. Pour résoudre ce problème, ouvrez vos paramètres d'affichage et désactivez la fonction de rotation.  Contactez votre fabriquant de carte graphique pour plus d'informations concernant cette option de rotation.

- DirectX 9.0c est inclus dans le CD 1 de L.A. Street Racing, mais vous aurez peut-être également besoin de télécharger les derniers pilotes compatibles DirectX 9.0 sur le site Internet de votre fabriquant de carte graphique. De nombreux problèmes techniques peuvent ainsi être évités simplement en mettant à jour vos pilotes de carte graphique et de carte son.

- Selon la configuration de votre système, les performances du jeu peuvent changer du tout au tout en modifiant les options graphiques dans le menu Paramètres vidéo. Réduire la résolution et le mode vidéo peut permettre d'améliorer la vitesse globale du jeu, tandis que si vous augmentez ces paramètres le jeu sera probablement plus beau mais aussi plus lent. Si les performances du jeu sont toujours réduites, n'oubliez pas que vous pouvez également baisser le niveau de détail et désactiver les ombres sur ce même écran. Pensez à cliquez sur Appliquer après avoir modifié des options dans le menu. Régler une
(ou plusieurs) options vidéo peut améliorer radicalement la vitesse du jeu.

- Pour des performances optimales nous vous conseillons de fermer tous les autres programmes ou applications d'arrière-plan avant de jouer à L.A. Street Racing.

9. Trucs et astuces
=========================

1.  Si vous terminez une ZONE INTACTE, vous gagnerez une seconde de Nitro.
2.  Si vous terminez un TOUR INTACT, vous gagnerez 5 secondes de Nitro.
3.  Gardez toujours un oeil sur votre adversaire, il peut utiliser des raccourcis !
4.  Vous pouvez gagner un nouveau véhicule uniquement si la voiture que vous utilisez est de Niveau 1.
5.  Si vous voulez progresser rapidement, défiez les adversaires qui ont un prestige élevé.
6.  Utilisez le frein à main pour tourner dans des virages serrés et passer des obstacles.
7.  Les équipements qui proviennent d'un véhicule d'une catégorie élevée rapportent plus de points de prestige.  Changez de voiture dès que vous le pouvez. 
8.  Les équipements 'Puissance et Maniabilité' servent à améliorer la conduite de votre voiture.
9.  Gardez toujours un oeil sur le classement.
10. Vous pouvez défier un adversaire en cliquant sur son nom dans le tableau des classements. 
11. Toutes les voitures se conduisent différemment. Trouvez celle qui correspond le mieux à votre style !
12. Si vous voulez progresser rapidement, pensez à parier sur tous les équipements dans l'écran des défis.
13. Essayez de toujours garder un peu de Nitro pour la fin de la course !
14. Une boîte de vitesse automatique peut devenir bien plus efficace en passant les vitesses manuellement.
