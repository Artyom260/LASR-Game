2. Zum Einstieg
=========================

Schließen Sie alle laufenden Programme (Anti-Virus, E-Mail-Client, Browser, File Sharing, Instant Messaging), bevor Sie L.A. Street Racing installieren. Die meisten Installationsprobleme können vermieden werden, wenn Sie die laufenden Programme schließen.  

Legen Sie die L.A. Street Racing-CD in Ihr CD\DVD-Laufwerk ein, klicken Sie auf INSTALLIEREN und folgen Sie den Anweisungen auf dem Bildschirm, um die Installation abzuschließen.

Wenn AutoPlay nicht aktiviert ist oder kein Installationsbildschirm erscheint, befolgen Sie die folgenden Schritte:

1. Klicken Sie auf START.
2. Klicken Sie auf AUSFÜHREN.
3. Geben Sie D:\SETUP.EXE ein (wenn “D” Ihr CD\DVD-Laufwerk ist).
4. Folgen Sie den Anweisungen auf dem Bildschirm.

5. Problembehandlung
=========================

- Vergewissern Sie sich, dass Ihr Computer die minimalen Systemanforderungen erfüllt. Wenn Sie L.A. Street Racing auf einem Computer installieren, der nicht die minimalen Systemanforderungen erfüllt, kann dies zu nicht vorhersehbaren Ergebnissen führen.

- Schließen Sie alle laufenden Programme, vor allem Antiviren-Sofware, bevor Sie das Spiel installieren. Viele Installationsprobleme können dadurch
verhindert werden.

Sollten Sie weiterhin Probleme bei der Installation von L.A. Street Racing haben, befolgen Sie die folgenden Schritte:

1. Legen Sie CD 1 ein und doppelklicken Sie auf ARBEITSPLATZ.

2. Klicken Sie mit der rechten Maustaste auf das CD\DVD-Laufwerk und wählen Sie EXPLORER.

3. Gehen Sie auf BEARBEITEN und dann auf ALLES MARKIEREN.

4. Gehen Sie wieder auf BEARBEITEN und dann auf KOPIEREN.

5. Kopieren Sie die Dateien an einen Ort auf Ihrer Festplatte. Wir empfehlen Ihnen, sie in einen Ordner auf Ihrem Desktop zu kopieren. Klicken Sie dazu mit der rechten Maustaste auf Ihren Desktop, gehen Sie auf NEU und dann auf ORDNER.  Öffnen Sie den erscheinenden neuen Ordner, klicken Sie mit der rechten
   Maustaste in den Ordner und gehen Sie auf EINFÜGEN.

6. Die Dateien werden jetzt von CD 1 auf Ihre Festplatte kopiert.  Dies kann einige Minuten dauern.

7. Wenn das Spiel auf Ihre Festplatte kopiert ist, doppelklicken Sie in dem neuen Ordner auf RunSetup.exe, um das Spiel zu installieren.

Hierdurch wird das CD\DVD-Laufwerk aus dem Installationsprozess entfernt, sodass Probleme mit der CD ausgeschlossen werden.

- Wenn Sie mit Windows XP oder Windows 2000 arbeiten, müssen Sie L.A. Street Racing über einen Administrator- oder Power-User-Zugang installieren. Wenn Sie keine Administrator- oder Power-User-Rechte haben, kontaktieren Sie Ihren Systemadministrator.

- Wenn Sie eine Firewall oder einen Router haben, könnten Sie auf Schwierigkeiten stoßen, wenn Sie L.A. Street Racing über LAN oder das Internet spielen wollen.  Sie müssen die Firewall bzw. den Router so konfigurieren, dass L.A. Street Racing über Ports 443, 12321, 12346 Zugang zum Matchmaking-Masterserver hat.

- Vergewissern Sie sich, dass TCP und UDP für alle Ports zugänglich sind.  Sollten Sie damit Probleme haben, kontaktieren Sie Ihren Systemadministrator
oder den Verkäufer der Firewall bzw. des Routers.

- Wenn auf Ihrem Computer Windows XP mit Service Pack 1 (SP1)installiert ist und die Firewall zur Internetverbindung aktiviert ist, können Sie auf
Schwierigkeiten stoßen, wenn Sie L.A. Street Racing über LAN oder das Internet spielen wollen. Um Abhilfe zu schaffen, sollten Sie entweder das Service Pack 2 (SP2) installieren oder die Firewall zur Internetverbindung in Ihren Netzwerkeinstellungen deaktivieren. Befolgen Sie die folgenden Schritte, um die Firewall zu deaktivieren:

1. Klicken Sie auf Start.

2. Gehen Sie auf Systemsteuerung.

3. Gehen Sie auf Netzwerkverbindungen.

4. Klicken Sie mit der rechten Maustaste auf Ihr LAN-Icon und gehen Sie auf Eigenschaften.

5. Gehen Sie auf das Feld “Erweitert”.

6. Vergewissern Sie sich, dass die Funktion “Diesen Computer und das Netzwerk schützen, indem das Zugreifen auf diesen Computer vom Internet eingeschränkt oder verhindert wird.” nicht aktiviert ist.

- Wenn Sie das Windows XP Service Pack 2 (SP2) installiert haben, können Sie wegen der SP2-Firewall Schwierigkeiten haben, L.A. Street Racing über das Internet zu spielen. Um Abhilfe zu schaffen, befolgen Sie die folgenden Punkte:

1. Starten Sie L.A. Street Racing.

2. Klicken Sie im Hauptmenü auf Mehrspieler-Einstellungen.

3. Wenn der Serverbrowser erscheint, drücken Sie gleichzeitig ALT und TAB.  Damit gelangen Sie zurück zu Ihrem Windows-Desktop.

4. Das Dialogfeld „Windows-Sicherheitswarnung“ müsste jetzt erscheinen.  Klicken Sie auf „Nicht mehr blocken“, damit L.A. Street Racing Zugang zu den Spieleservern im Internet hat.

5. Klicken Sie in der Windows-Taskleiste auf L.A. Street Racing.  Damit kehren Sie zurück zum Spiel.

6. Verlassen Sie L.A. Street Racing, indem Sie im Hauptmenü auf Spiel verlassen gehen.

7. Starten Sie L.A. Street Racing neu. Jetzt müsste es Ihnen möglich sein, Zugang zu Spielen im Internet zu bekommen.

- Zumeist in Laptops gibt es Grafikkarten mit einer Drehfunktion, die bewirkt, dass das Spiel seitlich gedreht abläuft. Sie können das Problem lösen, indem Sie unter Anzeige und Eigenschaften die Drehfunktion deaktivieren. Kontaktieren Sie den Hersteller der Grafikkarte für weitergehende Informationen.

- DirectX 9.0c befindet sich auf CD 1 von L.A. Street Racing.  Es ist aber möglich, dass Sie darüber hinaus die neuesten Treiber für DirectX 9.0 von der Website Ihres Hardware-Herstellers herunterladen müssen. Viele technische Probleme können gelöst werden, indem Sie die Treiber für Ihre Grafik- und Soundkarten aktualisieren.

- Je nach Systemkonfiguration kann das Gameplay verbessert werden, indem Sie die Grafikoptionen unter Einstellungen, Grafikeinstellungen ändern. Wenn Sie die Auflösung und den Grafikmodus herabsetzen, läuft das Spiel schneller.  Umgekehrt läuft das Spiel möglicherweise langsamer, sieht aber grafisch besser aus.  Sollte das Gameplay nach wie vor schlecht sein, können Sie auch die Detailstufe und die Schatten herabsetzen. Denken Sie daran, dass Sie Grafikänderungen bestätigen drücken, wenn Sie Änderungen vorgenommen haben. Wenn Sie eine (oder alle) dieser Grafikoptionen ändern, können Sie die
Spielgeschwindigkeit erheblich steigern.

- Für optimales Gameplay empfehlen wir Ihnen, alle anderen Programme und Anwendungen zu schließen, bevor Sie L.A. Street Racing spielen.

9. Tipps und Tricks
=========================

1.  Wenn du einen FEHLERFREIEN ABSCHNITT schaffst, erhältst du 1 Sekunde Nitro. 
2.  Wenn du eine FEHLERFREIE RUNDE schaffst, erhältst du 5 Sekunden Nitro. 
3.  Achte auf deine Gegner! Sie kennen möglicherweise Abkürzungen! 
4.  Du kannst nur ein neues Auto gewinnen, wenn dein derzeitiger Wagen mind. Stufe 1 ist. 
5.  Wenn du schneller vorankommen willst, fordere Gegner mit höherem Prestige heraus. 
6.  Benutze die Handbremse, um durch enge Kurven oder an Gegenständen vorbeizufahren.
7.  Gegenstände, die zu einem Auto mit höherem Wert gehören, sind mehr Prestigepunkte wert. Wechsle das Auto, wenn du kannst. 
8.  Wenn du Stärke und Handhabung aufrüstest, verbessert sich das Fahrverhalten deines Autos. 
9.  Behalte stets ein Auge auf die Rangliste. 
10. Du kannst deine Gegner herausfordern, indem du auf ihren Namen in der Rangliste klickst.
11. Alle Wagen sind unterschiedlich in der Handhabung. Suche dir das Auto, das am besten zu deinem Rennstil passt!
12. Wenn du schneller vorankommen willst, wette um alle Gegenstände auf dem Herausforderungs-Bildschirm.
13. Behalte immer ein bisschen Nitro für das Ende des Rennens!
14. Du kannst das Automatikgetriebe effektiver machen, wenn du manuell herauf- und herunterschaltest.
